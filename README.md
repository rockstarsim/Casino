# Casino Royale

Multiplayer online casino with **Blackjack**, **Baccarat**, and **Texas Hold'em**.

## Play

1. Start the server: `npm install && npm start`
2. Open http://localhost:3000
3. Create a room or join with a 6-character code
4. Share the room code with friends!

## Games

- **Blackjack** — Up to 7 players vs the dealer
- **Baccarat** — Up to 9 players bet on Player, Banker, or Tie
- **Texas Hold'em** — 2–9 players with blinds and betting rounds

## Deploy to Render (free cloud hosting)

1. Push this repo to GitHub
2. Go to [render.com](https://render.com) → New → Blueprint
3. Connect your GitHub repo — Render reads `render.yaml` automatically
4. Your live URL will be `https://casino-xxxx.onrender.com`

## Tech

- Node.js + Express + Socket.io
- Vanilla HTML/CSS/JS frontend
