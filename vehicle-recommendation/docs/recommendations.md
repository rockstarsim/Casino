# 실전 추천 요약

순위표 상위권은 TCO·연비 비중이 커서, **승차감·재난·희소성·120kg·공회전**까지 반영한 실전 추천입니다.

## TOP 5

### 1. 토요타 RAV4 HEV 4WD (2019~22, 2,400~2,800만)

- 감가 방어, 4WD·지상고, 고속 안정성, 짐·출장 균형
- ACC·통풍·오토홀드는 **XLE/Limited급** 트림 확인

### 2. 렉서스 NX350h AWD (2021~23, 3,000~3,400만)

- 승차감·감가·10년 보유 최적, RSPA 등 주차 보조 우수
- 할부 필요

### 3. 혼다 CR-V HEV AWD (2023~25, 2,900~3,300만)

- 실내·트렁크 넓음, RAV4 대비 덜 흔함

### 4. 토요타 RAV4 PHEV AWD (2022~24, 3,200~3,600만)

- **업무 대기·음악**에 최적 (EV 모드 공회전), V2L

### 5. 기아 셀토스 HEV 시그니처 (2026, 3,470만)

- 신차·V2L·연비 19.5km/L·주차 용이, 다소 흔한 점은 감점

## 피할 선택

- 스포티지·투싼·테슬라 신차 (너무 흔함)
- 레이/캐스퍼급 단순 교체 (요구사항 미충족)
- 아파트 충전 없이 EV만 고집 (감가·TCO 리스크)

## 최종 한 줄

> **1,700만 예산 + 10년 + 승차감 + 재난** → RAV4 HEV 4WD  
> **할부 3,000만 OK** → NX350h 또는 CR-V HEV  
> **공회전·대기 최우선** → RAV4 PHEV AWD
