# 구매자 프로필 및 조건

## 기본 정보

- 거주: 인천
- 연령/성별: 30대 초반 남성
- 체구: 120kg
- 월 수입: 약 250만원
- 차량 예산: 현금 1,000만원 + 레이 매각(~700만원) ≈ **1,700만원**

## 현재 차량

- **기아 레이** 2013년식, 134,000km, 시세 약 700만원
- 평균 연비 7.5km/L
- 불만: 승차감, 고속 안정성, 옵션 부족

## 주행 패턴

- 수도권 주간 100~150km (연 약 6,500~7,800km)
- 왕복 300km 출장 연 3~4회
- 국내 여행 증가 예정
- **추정 연간 주행: 약 13,000km**

## 필수 조건

1. 어댑티브 크루즈 컨트롤 (ACC)
2. 오토 홀드
3. 통풍 시트
4. N(중립) 주차 가능
5. 유지비 저렴
6. 재난(지진·홍수·폭설·전쟁) 대응 용이
7. 짐 적재 많이, 주행 성능 빠릿함
8. 120kg 체구에 불편하지 않을 것
9. 주차 용이 (크면 자동 출차 필수)
10. 공회전에 용이 (차 안 대기·음악 청취 시간 김)

## 선호·제외

| 선호 | 제외 |

| 흔하지 않은 차 | 너무 흔한 차 (스포티지·투싼·테슬라 등) |
